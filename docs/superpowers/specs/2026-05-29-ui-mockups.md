## 组件库复用规范

### 通用按钮

| 类型 | 样式 |
|------|------|
| Primary | 背景 `#FF6B35`，文字白色，圆角 22px |
| Secondary | 背景 `#004E89`，文字白色，圆角 8px |
| Outline | 透明背景，边框 `#FF6B35`，文字 `#FF6B35` |
| Ghost | 透明背景，无边框，文字 `#FF6B35` |

### 通用卡片

| 类型 | 样式 |
|------|------|
| 标准卡片 | 白色背景，圆角 8px，阴影 sm，内边距 16px |
| 强调卡片 | 主色背景 `#FF6B35`，文字白色 |
| 信息卡片 | 浅灰背景 `#F5F5F5`，圆角 8px |

### 通用状态色

| 状态 | 颜色 |
|------|------|
| 成功 | `#4CAF50` |
| 警告 | `#FF9800` |
| 错误 | `#F44336` |
| 信息 | `#2196F3` |
